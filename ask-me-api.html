Ask Me API

Have all your questions about any OAS answered using the chatbot

  1. Load the link to your Open API Specifications
    • eg – "Load https://petstore3.swagger.io/api/v3/openapi.json"
  2. Start asking your questions!
  3. Try saying “How many endpoints does it have?”
    • or “What are the API’s endpoints?”
    • Get more specific with an endpoint –
      • “Describe <endpoint>”
      • and then “What are <endpoint>’s http methods?”
      • or “Give me <http_method>’s security information”
